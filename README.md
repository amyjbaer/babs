# B.A.B.S. - Balanced Algorithmic Blocking System

A sophisticated dice rolling application with intelligent seven-frequency control.

## Features

- **Sliding Window Control**: Limit the number of sevens within a rolling window of rolls
- **Overall Percentage Control**: Keep seven frequency at or below a target percentage
- **Visual Effects**: Animated dice rolls, blocking effects, and celebration animations
- **Detailed Statistics**: Track roll history, distribution, and prevention metrics

## How It Works

B.A.B.S. uses advanced algorithms to control the frequency of rolling sevens (the most common result in two-dice rolls). Configure your preferred settings and watch as the system intelligently manages seven occurrences while maintaining randomness for all other outcomes.
